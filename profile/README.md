# Мой МПТ

**Мой МПТ** — мобильное приложение для студентов Московского приборостроительного техникума им Г.В. Плеханова, позволяющее просматривать расписание занятий, звонки и другую полезную информацию.

## Навигация

* [Основной репозиторий «Мой МПТ».](https://github.com/AppMyMPT/MyMPT)
* [Репозиторий с парсером для «Мой МПТ».](https://github.com/AppMyMPT/ReplacementVPS)
* [Архивный репозиторий «Мой МПТ».](https://github.com/Merrcurys/MyMPT)
